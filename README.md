# 🏨 Hotel System

Backend system for hotel management built in Python.

## Structure
```
Hotel/
├── logs/
│   └── hotel.log
├── models/
│   ├── persona.py        # Person, Huesped, Trabajador
│   ├── habitacion.py     # Habitacion, Suite, Master, SuitePresidencial
│   ├── reservacion_hotel.py
│   └── reservacion_restaurante.py
├── errors/
│   └── hotel_errors.py
├── app.py
└── logs.py
```

## Features
- OOP with inheritance and composition
- Custom error handling
- Professional logging
- Type hints
- SOLID principles

## Tech
- Python 3.10+
